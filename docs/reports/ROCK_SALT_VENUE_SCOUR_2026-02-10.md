**Finding:** [24tix.com](https://www.24tix.com) is the **master ticketing platform** for Utah’s indie music scene. It aggregates events from multiple promoters and venues in one place.

**Promoters / sources on 24tix:** S&S Presents (Kilby Court, Urban Lounge), JRC Events (metal/rock), Quarters Arcade Bar, Alternative Arts and Music Program, Sequoia Productions, KRCL Presents, and many venue-specific listings.

## Putting events on TheRockSalt.com

To load these events onto the live site so they appear on the [events calendar](https://www.therocksalt.com/events), use the ingest script and payload in the **rock-salt** app: see **`apps/rock-salt/docs/IMPORT_SCOUR_EVENTS.md`**. Summary: run the migration that allows `venue_scour` as an external source, set `CRON_SECRET`, then run `pnpm import-scour-events` (with the app or `ROCK_SALT_API_URL` pointing at production).
