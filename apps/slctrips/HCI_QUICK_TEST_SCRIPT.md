# HCI Quick Test Script - Rapid Evaluation

**Date:** January 2025  
**Purpose:** Quick 15-minute evaluation of key features  
**Method:** Use Claude Chrome Extension

---

## ⚡ 5-MINUTE QUICK TEST

### Test 1: Morbid Misdeeds TripKit (2 min)

**URL:** `/tripkits/morbid-misdeeds/view`

**Claude Prompt:**
```
Quick test - verify these 5 things:

1. Are 5 deep dive stories visible? (Ted Bundy, Elizabeth Smart, etc.)
2. Can you click a story and see full content?
3. Is there a Share button in Actions section?
4. Does the progress bar have a shimmer effect?
5. Do story cards lift on hover?

Answer: Yes/No for each, and note any issues.
```

---

### Test 2: Welcome Wagon (2 min)

**URL:** `/welcome-wagon/week-one-guide`

**Claude Prompt:**
```
Quick test - verify:

1. Does progress bar have shimmer animation?
2. Is ShareButton in actions bar?
3. Check off an item - does progress update?
4. Reach 100% - does celebration appear?
5. Can you share the page?

Answer: Yes/No for each.
```

---

### Test 3: Sharing Test (1 min)

**Any TripKit or Welcome Wagon page**

**Claude Prompt:**
```
Test sharing:

1. Click Share button
2. Does dropdown show Twitter, Facebook, LinkedIn, Email?
3. Click "Copy Link" - does it work?
4. Test one platform (Twitter) - does it open?

Report: Working / Not Working / Issues
```

---

## ✅ QUICK VERIFICATION

**All Good If:**
- ✅ Stories visible in TripKit viewer
- ✅ Sharing works on all pages
- ✅ Animations smooth
- ✅ Progress celebrations work
- ✅ No errors in console

**Issues If:**
- ❌ Stories missing
- ❌ Sharing broken
- ❌ Animations janky
- ❌ Errors in console
